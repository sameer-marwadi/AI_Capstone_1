# AI_Capstone_1
This folder contains a full, portfolio-ready data science project analyzing US traffic accidents (2016–2023).

## Contents

- `Capstone1_US_Accidents_Analysis_SM_12032025.ipynb` – Main Jupyter notebook:
  - CRISP-DM structure
  - Data cleaning & feature engineering
  - EDA, correlation heatmaps, geospatial scatter, time trends
  - Statistical tests (t-test, chi-square)
  - Predictive modeling (Random Forest, optional XGBoost)
  - Business insights and recommendations


- `requirements.txt` – Suggested Python dependencies.

## How to Run

1. Download the **US Accidents dataset** from Kaggle:
   https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents

2. Place `US_Accidents_March23.csv` in this project folder.

3. Create and activate a virtual environment (optional but recommended).

4. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

5. Launch Jupyter Notebook:

   ```bash
   jupyter notebook Capstone1_US_Accidents_Analysis_SM_12032025.ipynb
   ```

6. Launch Tableau Public Dashboard -https://public.tableau.com/views/US_Accidents_SM_12122025/Main?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link
7. Refer to PPT in Data Folder- US_Accidents_Presentation for imporant Data Insights and additional Deep Dive Analysis
